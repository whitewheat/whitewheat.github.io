数组 Array

概念相关

  • 数组名
  • 元素（值）
  • 角标、下标、索引
  • 数组长度 （元素个数）

数组特点

  • 数组是有序排列的
  • 数组属于引用类型的变量。数组的元素可以是基本数据类型，也可以是引用数据类型
  • 创建数组对象会在内存中开辟一整块连续的空间
  • 数组长度一旦确定，就不能修改

一维数组的使用

  • 一维数组的声明和初始化 （静态与动态）

        public static void main(String[] args) &#123;        
        //  初始化
        int[] array;//声明
        array=new int[] &#123;1,2,3&#125;;//初始化
        //1 静态初始化：声明+初始化 初始化和赋值同时进行完
        int[] arrayy=new int[] &#123;1,2,3&#125;;    
        //2 动态初始化 初始化和赋值不同时进行

  • 如何调用数组指定位置的元素 ( 数组名[索引] )

  • 如何获取数组的长度 (数组.length)

  • 如何遍历数组 （循环）

  • 数组元素的默认初始化值 （在动态初始化声明数组时，数组会根据数组长度在内存中分配一定数量的数组元素类型的默认值。 不会出现只声明一个基本数据类型后不赋值报错的情况）

    Java变量的默认初始化值

    类型 默认值
    boolean false
    byte short int long 0
    float double 0.0
    char \u0000 或者ASCII 0 不是‘0’
    对象 null

数组名就是该数组的地址

int[] arr1 = new int[]&#123;1,1&#125;;
        System.out.println(arr1[0] + "  " + arr1[1] );//  1   1

        int[] arr2 = new int[2];
        arr2 = arr1; //赋值后 arr2在栈中所存储的就是arr1所存储的地址
                    //现在 arr2 和 arr1 是两个不同的变量指向同一处地址 实际上是一个东西
        arr2[0] = 2;
        arr2[1] = 2;    //arr2 修改的是按地址找到的堆中的元素 

        System.out.println(arr1[0] + "  " + arr1[1]);//  2   2

二维数组

​ 一维数组的元素 类型可以为引用类型，当一维数组的元素类型又是一个数组 就是二维数组

//静态初始化
        int[][] arr1 = new int[][]&#123;&#123;1,2&#125; , &#123;3,4&#125; , &#123;5,6&#125;&#125;;
        //动态初始化1
        String[][] arr2 = new String[3][2];
        //动态初始化2
        String[][] arr3 = new String[3][];
